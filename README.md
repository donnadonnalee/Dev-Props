# DevToolbox

Engineer's Premium Utility Suite.

## Tools Included
- **Line Counter**: High-performance local directory analysis.
- **RegEx Tester**: Real-time regex testing with visual feedback and capture groups.
- **CSV Viewer**: Fast, high-density CSV viewer with sorting and search capabilities.

## Privacy & Security
All tools are purely static web applications. **Everything is processed locally in your browser.** 
None of your data (code, regex strings, CSV files) is ever uploaded to a server or stored externally.
